Presentaciones panorámicas desde tu Markdown

Inanna transforma textos en Markdown en presentaciones amplias listas para proyectar o imprimir. Controla la tipografía, los colores y la composición con plantillas diseñadas para narrativas visuales extensas, añade recursos multimedia, ajusta imágenes sobre la marcha y exporta a PDF conservando tu estilo de marca.

Desarrollado dentro del proyecto RuralNEXT de Fundación Repoblación, por Compañía Maximalista S.Coop.

Acceder a la aplicación

Funcionalidades

Editor Markdown

Escribe en texto plano, separa diapositivas con --- y obtén previsualizaciones instantáneas gracias a Parsedown. Guarda versiones en XML para reeditarlas siempre que quieras.

Plantillas panorámicas

Selecciona plantillas pensadas para pantallas anchas y combina contenidos con imágenes de forma equilibrada mediante un sistema de miniaturas y arrastrar y soltar.

Estilos personalizables

Define fuentes, colores y cajas destacadas para cada proyecto. Sincroniza con Google Fonts y conserva la identidad visual de tu organización.

Banco de recursos

Sube imágenes y vídeos, recórtalos y ajusta brillo o contraste antes de insertarlos en tus diapositivas. Gestiona todo desde un único panel.

Exportación a PDF

Genera PDFs panorámicos con wkhtmltopdf, incrustando fuentes y recursos para obtener resultados fieles a tu diseño original.
